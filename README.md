# Frontend Portfolio

Personal one-page frontend portfolio built with React 18 and TypeScript.
The portfolio showcases selected projects, skills, and experience as a frontend developer.

## Tech Stack
- React 18
- TypeScript
- Vite
- CSS Modules
- Framer Motion (optional)

## Features
- One-page scroll layout
- Anchor-based navigation
- Responsive design
- Clean and minimal UI
- Accessible keyboard navigation

## Live Demo
https://your-domain.com

## Projects
Each project displayed in the portfolio has:
- A short description of the problem it solves
- Tech stack overview
- Link to a live demo
- Link to the GitHub repository
